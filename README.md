This project demonstrates Model Validation, Cross-Validation, and Hyperparameter Tuning using the California Housing dataset. The goal is to improve machine learning model performance by evaluating different regression algorithms and optimizing model parameters.

The project compares:

Linear Regression
Ridge Regression
Decision Tree Regressor
Tuned Decision Tree Regressor (using GridSearchCV)

It also applies feature scaling, train-test splitting, cross-validation, and hyperparameter tuning to achieve better prediction accuracy.

🎯 Objectives
Load and preprocess the California Housing dataset.
Perform feature scaling using StandardScaler.
Train multiple regression models.
Evaluate model performance using:
RMSE (Root Mean Squared Error)
R² Score
Apply Cross-Validation for reliable model assessment.
Use GridSearchCV to find the best hyperparameters.
Compare model performances.
